# SBG Database Setup

SBG supports multiple database backends. Set the corresponding environment variable to enable:

- `MONGO_URL`: MongoDB Connection String
- `SUPABASE_URL` & `SUPABASE_KEY`: Supabase project
- `NEON_URL`: Neon Postgres connection
- `APPWRITE_URL` & `APPWRITE_KEY`: Appwrite Project
- `SQLITE_URL`: Local SQLite file path
- `MYSQL_URL`: MySQL Connection URL
