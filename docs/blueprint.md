# **App Name**: SBG

## Core Features:

- Multi-Provider Database Sync: Automatically detect and connect to MongoDB, Supabase, Neon, or SQL systems for persistent data management.
- Dynamic Session Handshake: Encoded Base64 session handling that supports custom prefixes like SBG~ or ASTRAX~ for zero-hardcode authentication.
- Modular Command Engine: A plugin-based router that dynamically loads general, event-based, and anti-system handlers.
- Intelligent Agent Flow: A generative AI flow tool that reasons over user intents to route incoming WhatsApp messages to specific logic blocks.
- Public Presence Dashboard: A Next.js hosted status portal displaying bot health, connection stability, and real-time session logs.
- Structural Message Formatter: Enforces a consistent boxed-style visual architecture for all bot responses and first-connect notifications.
- Autonomous Resource Janitor: Automated background service that monitors RAM usage and cleans orphaned sessions to ensure uptime.

## Style Guidelines:

- The palette is inspired by technical prestige and gold-standard intelligence. The primary color is a refined Gold (#E1B64E).
- Background: A deep, midnight obsidian (#141311) to maximize the contrast of gold UI elements.
- Accent: A sophisticated burnt terra-cotta (#CD5233) for critical system notifications and session alerts.
- Headlines and technical labels: 'Space Grotesk' for a computerized and high-tech feel.
- Body text: 'Inter' for clear, machined legibility.
- High-fidelity, monochromatic outline icons representing botanical and electrical nodes, aligned with the SBG theme.
- Rigid, geometric grid systems that reflect the structural integrity of the 'box-format' message protocol.
- Micro-pulse transitions on connectivity icons to signal live socket status without being distracting.
